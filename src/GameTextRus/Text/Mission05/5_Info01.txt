<P><JC>Проект Титан</JC>
<P><JC>Краткое описание</JC>
<P>
<P>Проект Титан - самый успешный прототип искусственного разума, созданный на сегодняшний день. Система состоит из трех отдельных аналитических блоков, обрабатывающих информацию по своему собственному алгоритму. Это позволяет принимать наиболее правильные решения в любой ситуации, т.к. конечный результат представляет собой усредненный вариант ответа на поставленную задачу каждым из трех блоков.
<P>Ниже приводится подробное описание каждого из трех блоков.
<P>Гиперион. Сердце системы. Именно Гиперион дает Титану то, без чего он бы не смог существовать: разум. Сегмент основывается на мудрости и правильности своих решений. Гиперион способен решать самые сложные задачи с минимальной потерей для обеих сторон. Этот сегмент был установлен в первую очередь. Гипериону не свойственно решать вопросы с выгодой только для одной стороны. Это делает его наиболее контролируемым блоком Титана.
<P>Хронос. Решения этого сегмента принимаются на основе анализа последствий, которые могут последовть после ответа на текущий запрос. Иными словами, сегмент Хронос оценивает последствия своих решений. Этот сегмент был установлен в систему вторым.
<P>Крий. Мыслительный процесс сегмента основывается на максимально правильном решении относительно личной выгоды (или же обоюдной выгоды с приближенным к Титану объекту). Сегмент Крий был устаовлен в самую последнюю очередь, так как никто не мог предугадать, что может случиться с сознанием Титана и какие решения он будет принимать, если на первом месте для него будут собственные амбиции.
